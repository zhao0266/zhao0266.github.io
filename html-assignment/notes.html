Simple Introduction to HTML Tags

"h1"
This tag represents the heading of the highest level in HTML.
"p"
This tag is used to define a paragraph in HTML.
"a"
This tag is used to create hyperlinks in HTML.
"img"
This tag is used to insert images into an HTML document.
"div"
This tag is a container element used to group and divide sections of content in HTML.
